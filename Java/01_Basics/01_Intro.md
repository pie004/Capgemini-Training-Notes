## 📘 Introduction to Java

* **Java** is a **high-level, object-oriented, class-based programming language**.

* It is designed to be:

  * Simple
  * Secure
  * Portable
  * Robust

* **Uses of Java:**

  * Desktop applications
  * Web applications
  * Enterprise systems
  * Mobile applications (Android)
  * Embedded systems

* **History:**

  * Developed by **James Gosling** and his team at **Sun Microsystems** in **1995**
  * Currently owned by **Oracle Corporation**

---

## 📘 Sample Java Program Structure

class ClassName
{
    public static void main(String[] args)
    {
        // Statements
    }
}

## 🔑 Important Points

* Every Java program must have a **class**.
* Execution starts from the **main() method**.
* `public static void main(String[] args)` is the **entry point** of a Java program.
* Statements are written inside the `main` method.
